# A-alogithm-study
A형 코딩테스트 합격을 목표로 하는 알고리즘 문제 풀이 스터디 입니다.

## ⚙️ 환경 설정
- IDE: Eclipse
- Language: Java 8

---

## 📌 스터디 진행 규칙

### 1. 문제 진행 방식
- 휴일(공휴일, 빨간날)마다 2문제씩 진행
- 강사님이 제공한 A형 역량 테스트 문제 기준
- 목록 상단부터 순서대로 2문제씩 풀이

---

### 2. 브랜치 & PR 규칙
- 개인 브랜치에서 문제 풀이
- 풀이 완료 후 PR로 main 브랜치에 병합
- PR 제목 형식

  - [BOJ] 2470 두 용액 - 투 포인터 - (본인이름)

  - [SWEA] 2115 벌꿀채취 - (본인이름)

---

### 3. PR 작성 규칙
- 사용 알고리즘
- 핵심 풀이 포인트 또는 간단 설명
- (선택) 어려웠던 점 / 실수 포인트

---

### 4. 풀이 시간 규칙 
- 문제당 최대 2시간
- 시간 초과 시에도 제출 필수
- 접근 시도 / 실패 원인 / 다시 볼 포인트 정리

---

## 🎯 스터디 목표
- A형 코딩테스트 대비 실력 확보
- 문제 접근 사고력 강화
- 혼자서도 해결 가능한 문제 유형 확장
